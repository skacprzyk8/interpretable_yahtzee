Code and notebooks supporting the Master's thesis "Interpretable Strategies for Yahtzee: An Imitation-Learning Approach".

## Contents
- `implementation.ipynb` — main experiments and models  
- `data analysis.ipynb` — evaluation and plots  
- `generation.py`, `dagger_execution.py` — data generation / simulation scripts 
- `Extracted trees and decision paths/`, `Images/` — saved outputs and figures.

1. Install ....
